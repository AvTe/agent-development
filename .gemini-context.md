# Gemini Agent Context for JobHunter AI

This file provides context to the Gemini CLI agent to help it understand the project and assist with development tasks.

## Project Overview

JobHunter AI is a Next.js application designed to help users manage their job applications efficiently. The application follows a data-driven workflow:
1. **Resume Management**: Users can upload PDF resumes which are parsed into structured JSON data using AI.
2. **Job Analysis**: Users can paste job descriptions to extract key requirements and responsibilities.
3. **Smart Matching**: The system compares resumes against job descriptions to provide match scores, gaps, and suggestions.
4. **Email Generation**: AI generates personalized, tone-aware application emails based on the match analysis.
5. **Job Search**: Integrated job search functionality powered by SerpApi (Google Jobs engine).

## Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** JavaScript
- **Styling:** Tailwind CSS
- **Database:** SQLite (using `better-sqlite3`)
- **AI Integration:** NVIDIA API using `moonshotai/kimi-k2.5` model for parsing, analysis, and generation.
- **PDF Parsing:** `pdf-parse` for text extraction from resumes.
- **Search Integration:** SerpApi for job search functionality.
- **Infrastructure:** Local file-based storage for database and uploads.

## How to Run the Project

- **Development:** `npm run dev`
- **Build:** `npm run build`
- **Start:** `npm start`
- **Environment Variables:** Requires `NVIDIA_API_KEY` and `SERPAPI_KEY` in `.env.local`.

## Database Schema

The database schema is defined in `db/database.js` and consists of four tables:

- **`resumes`**: Stores uploaded resume information.
  - `id`: INTEGER (Primary Key)
  - `filename`: TEXT
  - `raw_text`: TEXT
  - `parsed_data`: TEXT (JSON)
  - `created_at`: DATETIME

- **`jobs`**: Stores analyzed job descriptions.
  - `id`: INTEGER (Primary Key)
  - `title`: TEXT
  - `company`: TEXT
  - `raw_description`: TEXT
  - `parsed_data`: TEXT (JSON)
  - `created_at`: DATETIME

- **`matches`**: Stores the results of matching a resume to a job.
  - `id`: INTEGER (Primary Key)
  - `resume_id`: INTEGER (Foreign Key)
  - `job_id`: INTEGER (Foreign Key)
  - `score`: INTEGER
  - `analysis`: TEXT (JSON)
  - `created_at`: DATETIME

- **`emails`**: Stores generated application emails.
  - `id`: INTEGER (Primary Key)
  - `match_id`: INTEGER (Foreign Key)
  - `subject`: TEXT
  - `body`: TEXT
  - `tone`: TEXT
  - `created_at`: DATETIME

## Core Logic (lib/)

- `resumeParser.js`: Uses AI to extract details like skills, experience, and education from raw text.
- `jobAnalyzer.js`: Uses AI to extract title, company, skills, and responsibilities from job descriptions.
- `matcher.js`: Uses AI to compare candidate data against job requirements and generate a match score.
- `emailGenerator.js`: Uses AI and match analysis to create personalized, tone-specific emails.
- `kimi.js`: Axios client for communicating with NVIDIA's API.
- `database.js`: SQLite connection and schema initialization logic.
